# 扩展介绍

纯粹的游戏启动页美化扩展，仿三国杀一将成名界面

## 三步玩转【雷霆】

1、在扩展的设置中添加皮肤或露头皮肤（若有需要）的文件夹，可外链其他扩展的文件夹；
2、添加动态皮肤文件夹，可外链其他扩展文件夹；
3、添加动皮播放参数文件（如dynamicSkin.js），开玩！
