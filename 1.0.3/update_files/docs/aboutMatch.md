## 关于排位赛

这里讲述排位赛的玩法规则

## 基本规则

基本沿用三国杀十周年的排位规则，但做了部分简化，具体为：

等阶简化为：卫士 → 校尉 → 中郎 → 领军 → 大将 → 枭雄。

枭雄之前每段位分5段：
卫士（最低：卫士五 → 卫士四 → 卫士三 → 卫士二 → 卫士一）
校尉（校尉五～一）
中郎（中郎五～一）
领军（领军五～一）
大将（大将五～一）

赢一局+1星，失败-1星。升段所需星数
卫士（3星）
校尉（3星）
中郎（5星）
领军（5星）
大将（5星）

失败掉段规则：当前段掉到0星再失败会掉到上一阶段的2星或4星，如卫士三0星失败掉到卫士四2星。

枭雄不再晋级段位，星数最高累计到200星，枭雄0星失败不会掉到大将。

## 赛季规则

每一个自然月为一个赛季，赛季结束将会进行段位重置，具体规则：

卫士阶段不管几级赛季重置只到当前阶段0星，如卫士四3星重置为卫士四0星。

校尉以上都重置为上一个小阶段的0星（除校尉五，校尉不会重置为卫士阶段）例如校尉四3星重置为校尉五0星，大将一3星重置为领军五0星，枭雄180星重置为大将五0星。

## 其他

目前排位赛支持三种模式：身份场、对决和国战（其中对决将人数设置为4人即为十周年的2V2模式）

三种模式分别积分互不干扰。
