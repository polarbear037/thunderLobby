## 启动页和大厅页背景动画

这里讲述如何更换喜欢的登录页和大厅页动画

## 基本方法

扩展自带2套动态背景（曹金玉-瓷语青花、神赵云=百战金甲）作为兜底资源，严禁删除或随意修改路径，有可能会导致扩展启动异常！

可查看并参考配置文件[assetsPath.json](../config/assetsPath.json)中的写法（基本与十周年UI的配置方法相似，不同处如下：

1、没有"chuchang"属性，取而代之的是"action"属性可设数组，播放时会顺序播放action数组中的动作，并停留在最后一个动作loop；

2、新增"click"属性，用于在“醇享模式”中通过点击屏幕播放指定动作，同样支持数组。

3、新增"qianjing"属性，适用于有2层动态背景的皮肤。

4、”x“、"y"、"scale"属性同时支持数值写法和数组写法，"x":0.5等价于"x":[0,0.5]，"scale"写为数组可分别控制x轴和y轴的缩放；”x“、"y"、"scale"也可以省略不写，默认值为0.5。

5、当你不知道动画文件具体有什么"action"时，"action"属性也可以省略，程序会自动寻找合适的"action"进行播放。

## 扩展方法

扩展允许用户添加自己喜爱的动画作为启动页和大厅页背景动画。

在首次运行时会检查扩展根目录下是否有customAssets.json文件，若没有会自行创建，用户可自行在里面添加内容，不用担心会被更新覆盖（更新文件不会包含customAssets.json）。内容如下：

```json
{
  "pageSpine": {
    "spines": [ ]
  }
}
```

请注意添加的动画要在"spines"属性后的数组中括号内。

customAssets.json文件中的动画配置文件写法与assetsPath.json完全相同，并支持外链至其他扩展文件夹中的spine文件，具体写法如下（以链至十周年UI资源文件夹为例）：

```json
{
    "title": "鲍三娘-兔娇春浓",
    "name": "/extension/十周年UI/assets/dynamic/鲍三娘/兔娇春浓/daiji2",
    "x": 0.5,
    "y": 0.5,
    "scale": 0.6,
    "beijing": {
        "name": "/extension/十周年UI/assets/dynamic/鲍三娘/兔娇春浓/beijing",
        "scale": 0.3,
        "action": "play",
        "x": 0.5,
        "y": 0.5
    }
}
```

添加多个动画时一定要注意json文件的书写规则，每个大括号之间要用逗号分隔，且最后一个动画属性大括号之后不能带有逗号，动画配置的属性要加引号""，同样最后一条属性不得带逗号结尾

## 醇享模式

在大厅页面点击右上角“醇享模式”按钮进入。此时点击屏幕将会依次播放用户指定的action，点击右上角退出可退出该模式。
